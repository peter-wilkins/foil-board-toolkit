# Product Brief

## Elevator Pitch

Foil Board Toolkit is an open-source parametric design generator for DIY foil board builders. Instead of tracing an existing board, builders describe the rider, foil, discipline, and target behaviour, then get an original board design they can inspect, tune, and eventually machine.

## Why This Matters

DIY foiling has strong workshop energy but weak shared design tooling. Home builders can find build logs, forum advice, and finished board dimensions, but there is no obvious open design system that turns rider/use-case constraints into a coherent board shape.

This project fills that gap by making the design model explicit.

## First Users

- Peter, building boards on a 3 x 2 m CNC router.
- DIY wingfoil and downwind builders.
- Small composite workshops.
- Designers who want reproducible baselines rather than starting from blank CAD.

## Core Constraint

This must be an original generator, not a clone library. We can learn from public product trends, but we should avoid copying any one commercial board.

## First Success

A builder can generate a simple board outline/profile, look at it, adjust a few parameters, and export templates accurate enough to discuss or cut a test blank.
